# C言語
- 配列の宣言
    - データ型名 配列名[要素数];
        - int int_arr[5];
        - char char_arr[5];
    - 文字列の配列(文字列テーブル)
        - 文字列自体が配列なので結果的に2次元になる
        - char name[3][5];

- input
    - scanf
        - 文字列 %s char*

- 文字列の比較(strcmp)
    - #include<string.h>
    - strcmp(文字列1, 文字列2);
    - 戻り値
        - 第1引数 < 第2引数 の場合はマイナスの値
        - 第1引数 > 第2引数 の場合はプラスの値
        - 第1引数 = 第2引数 の場合は0
